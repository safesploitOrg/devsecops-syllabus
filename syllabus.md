## Syllabus Structure

- **Modules 01–03** establish DevSecOps, DevOps, and security foundations  
- **Modules 04–06** focus on platforms, pipelines, and infrastructure security  
- **Modules 07–10** address compliance, threat modelling, containers, and incident response  
- **Modules 11–13** focus on professional practice, hands-on projects, and portfolio positioning  

---

## Syllabus Overview

| Module | Focus Area                     | Core Outcomes                                                         |
|--------|--------------------------------|------------------------------------------------------------------------|
| 01     | DevSecOps Introduction          | Threat-aware delivery mindset and secure-by-design principles          |
| 02     | DevOps Fundamentals             | Automated delivery pipelines and infrastructure-as-code foundations    |
| 03     | Security Fundamentals           | Practical security controls and secure CI concepts                     |
| 04     | Tools & Technologies            | Understanding platform, container, and tooling trade-offs              |
| 05     | Secure SDLC                     | Security embedded into delivery workflows                              |
| 06     | Infrastructure Security         | Hardened cloud and hybrid infrastructure design                        |
| 07     | Compliance & Auditing           | Mapping real systems to regulatory frameworks                          |
| 08     | Threat Modelling & Risk         | Identifying trust boundaries and prioritising mitigations              |
| 09     | Secure Containers & Microservices | Secure containerised and distributed architectures                     |
| 10     | Incident Response & Recovery    | Detection, response, and learning from failure                         |
| 11     | Professional Practice           | Operating effectively across security, ops, and engineering            |
| 12     | Practical Projects              | End-to-end secure platform implementations                             |
| 13     | Portfolio & Career              | Translating execution into demonstrable capability                     |
